#  Iarlei Ferreira

**`Desenvolvedor FullStack`**

Sou Iarlei Ferreira de Barros, estudante de Ciência da Computação no IFCE – Campus Aracati e Técnico em Eletrotécnica. Tenho interesse em desenvolvimento Full Stack, com foco em Backend, buscando criar aplicações escaláveis, eficientes e bem estruturadas. Atualmente, venho aprimorando meus conhecimentos em Java, JavaScript, Node.js, Express, MySQL, SQL, Git e GitHub, além de fortalecer minha base em algoritmos, estruturas de dados e boas práticas de desenvolvimento de software. Gosto de transformar ideias em soluções práticas, sempre buscando escrever código limpo, aprender novas tecnologias e evoluir como desenvolvedor.
